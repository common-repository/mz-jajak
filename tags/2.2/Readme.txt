== Description ==

Welcome Mz-jajak WordPress Plugins.
This is one of the existing wordpress plugin. function to allow users of wordpress to add a facility poll on their blog as a widget.
To configure your poll, is very simple. You can go to the plugin options available, you can delete, edit, add and activate it. All of the data will be stored in a special database. For more information visit http://www.tirta-fajri.net

== Installation ==

1. Upload `Mz-jajak.php` and include directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the Mz-jajak Settings in your Administration Panel
4. Enable the Widget.
